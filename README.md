### 👾 &nbsp;about

Full-stack dev, 2–4 years deep. TypeScript and Rust when correctness matters, Python when I need to move fast against a model. Most of what I build stays offline — the 7% on GitHub is the stuff I decided the world could handle.

Currently all-in on **AI tooling infrastructure** under the [rift.sh](https://github.com/devflex-ai) org — a CLI agent, a developer IDE, and an AI platform. Same philosophy across all three: local-first, no fluff, built to last.
